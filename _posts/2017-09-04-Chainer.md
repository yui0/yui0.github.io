## chainerをインストール

```bash
# dnf install python-pip
$ pip install --upgrade pip
$ pip install chainer
```

## caffemodelをダウンロード

- 今回使用するモデルはGoogLeNet。[こちら](http://dl.caffe.berkeleyvision.org/bvlc_googlenet.caffemodel)からダウンロードする。

## chainerを使ってみる

```bash
$ pip install pillow
$ python predict.py
```
